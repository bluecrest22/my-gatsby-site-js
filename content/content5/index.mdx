## Next
***Next** is an open-source web development framework providing React-based web applications with server-side rendering and static website generation.*

 **Next  vs Gatsby**
 
 ***Next** should be used for building the following types of websites.
 -   *Client-side rendered applications*
-   *Big eCommerce websites*
-   *Finance websites*
-   *Web portals*
-   *Large multi-user websites*

**Gatsby** should be used for building the following types of websites.
-   *Static content websites*
-   *Highly secure websites*
-   *Progressive web apps*
-   *Documentation websites*
-   *Personal blogs or Portfolio websites*
