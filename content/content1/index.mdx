**Gatsby** *is an open source framework built on top of* **Node.js** *that utilizes* **React**, **GraphQL**, and **Webpack** *into a single tool for building static websites, and applications.* **Gatsby** *was launched in 2015, and is currently owned by Netlify a remote-first cloud computing company.*

 - **React** - *React is a free and open-source front-end JavaScript library for building user interfaces based on components.*
 - **Webpack** - *Webpack is a free and open-source module bundler for JavaScript.*
 - **Graph** - *GraphQL is an open-source data query and manipulation language for APIs and a query runtime engine.*
